# Full Stack Invoice CSV Import App

This project allows uploading CSV files with invoice data, storing them in a MySQL database via a Spring Boot backend, and viewing them in a paginated Angular frontend.

---

## Backend (Spring Boot + MySQL) â€“ Dockerized

> Make sure you have Docker and Docker Compose installed before proceeding.

### To start the backend:

```bash
docker-compose down
docker-compose up --build
```

## Frontend (Angular 20 + Tailwind CSS)

> Make sure Node.js (v18+) and npm are installed.

### Navigate to the frontend folder:

```bash
cd frontend
npm i
npm start

```
